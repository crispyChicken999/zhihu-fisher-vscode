# 智能伪装系统

智能伪装系统是知乎摸鱼的「防被发现」终极解决方案 —— 从详情页到侧边栏，全方位融入开发环境，让摸鱼天衣无缝！

---

## 🎭 详情页伪装

文章阅读页面失焦时自动伪装，获焦后自动恢复：

- **界面伪装** — 整个 WebView 切换为代码编辑界面，标题栏、图标全部替换
- **动态代码生成** — 实时生成 32 种编程语言的真实代码，语法高亮完美呈现
- **快捷键手动切换** — 按 **Space** 可手动切换伪装/恢复

## 📁 侧边栏伪装

将知乎列表侧边栏伪装成真实项目文件结构：

- **自动联动** — 打开文章或详情页伪装时自动启用
- **文件列表** — 根据选择的伪装类型生成对应的文件树
- **恢复方式** — 点击任意伪装文件即可恢复，或关闭全部知乎详情页后自动恢复
- **侧边栏菜单** — 通过侧边栏菜单可随时开启/关闭

## 💻 32 种伪装文件类型

| 类别 | 支持类型 |
|------|----------|
| **前端开发** | JavaScript、TypeScript、Vue、HTML、CSS、SCSS、LESS |
| **后端开发** | Python、Java、C#、PHP、Ruby、Rust、Swift、Lua、R |
| **系统开发** | C、C++（.cpp）、C++头文件（.hpp） |
| **脚本语言** | PowerShell、Shell |
| **数据格式** | JSON、XML、YAML、XSL |
| **配置文件** | INI、Git 配置文件 |
| **文档格式** | Markdown、LaTeX |
| **专用文件** | 应用日志、系统日志、TypeScript 定义（.d.ts）、Rust 工具链 |

## 🎯 自定义伪装类型

在阅读界面的设置面板 → **功能增强** Tab 中：
1. **自定义文件类型** — 勾选你希望伪装成的文件类型
2. **职业适配** — 例如前端工程师可以选择 JS/TS/HTML/CSS 等
3. **全选/全不选** — 方便快速配置
4. **预览效果** — 实时预览当前配置的伪装效果

> 如果不选择任何类型，则使用全部 32 种类型随机伪装。

## 🔧 如何启用

### 详情页伪装
- **默认开启** — 安装插件后即生效
- 在设置面板 **功能增强** Tab 中可开关
- 快捷键 **Space** 手动切换

### 侧边栏伪装
1. 确保详情页伪装已启用
2. 在侧边栏菜单中点击「智能伪装 → 侧边栏伪装 → 开启」
3. 或者在阅读界面设置面板中勾选「启用侧边栏伪装」

## 📊 查看状态

通过命令可以查看伪装系统的当前状态：
- `查看伪装缓存状态` — 显示启用状态、缓存数量和活跃 WebView
- 帮助您了解伪装系统是否正常工作

## ⚡ 技术原理

1. **页面监听** — 监听 VS Code 窗口和标签页的焦点变化
2. **状态检测** — 检测当前活动页面是否为知乎文章
3. **代码生成** — 失焦时随机选择文件类型，生成对应代码
4. **界面切换** — 替换标题、图标，渲染完整代码编辑界面
5. **语法高亮** — 应用 VS Code 原生语法高亮样式
6. **恢复机制** — 获焦时立即恢复原始内容

## 📌 注意事项

- **数据安全** — 伪装过程中原始文章保存在内存，不会丢失进度
- **性能影响** — 切换极快基本无延迟，资源占用极低
- **兼容性** — 支持 VS Code 1.82.0+，与其它插件无冲突
