# 文章阅读功能

在文章阅读界面，您可以享受丰富的交互功能和个性化设置。

## 🎯 沉浸模式

一键切换沉浸式全屏阅读，隐藏所有工具栏干扰，内容区域最大化。
- **快捷键：X** 快速切换
- 沉浸模式下在左侧可展开浮动工具栏
- 工具栏按钮可自定义隐藏/显示

## 🧭 回答导航

### 回答/文章切换
| 操作 | 快捷键 |
|------|--------|
| 上一个回答 | **←** 或 **A** |
| 下一个回答 | **→** 或 **D** |
| 上一篇内容 | **Ctrl+↑** 或 **W** |
| 下一篇内容 | **Ctrl+↓** 或 **S** |

### 页码跳转
- 当前回答索引下拉选择器，直接跳到指定回答
- 显示已加载回答数 / 总回答数
- 分页器支持快速跳转页码

### 回答排序
- **默认排序** — 按知乎默认热度排序
- **时间排序** — 按发布时间由新到旧排序
- 在问题页顶部点击排序按钮切换

## ⌨️ 快捷键一览

| 快捷键 | 功能 |
|--------|------|
| ← / → 或 A / D | 上/下一个回答 |
| Ctrl+↑/↓ 或 W/S | 上/下一篇内容 |
| **X** | 沉浸模式切换 |
| **。**（句号） | 打开设置面板 |
| **,**（逗号） | 展开/收起评论区 |
| **Space**（空格） | 代码伪装 |
| **G** | 灰色模式切换 |
| **/** | 切换图片显示模式 |
| **F** | 收藏到收藏夹 |
| **B** | 在浏览器中打开 |
| **C** | 复制文章链接 |
| **V** | 回到顶部 |
| **T** | 切换工具栏（沉浸模式） |

> 💡 所有快捷键均支持在设置面板中自定义！

## 🧠 知乎直答 AI 集成

无需离开 VS Code 即可使用知乎内置的 AI 功能：
- **AI 关键词解释** — 点击文章中的「知乎直答」链接，在弹窗中查看 AI 对关键词的解释
- **AI 总结回答** — 点击回答旁的 ✨ 按钮，让 AI 总结这篇内容讲了什么
- 通过 Puppeteer 模拟点击，内容自动搬运到 VS Code 弹窗中显示
- 完整免责声明：不破解/绕过任何知乎接口，不收集任何用户数据

## 📥 Markdown 导出

将当前文章/回答导出为 Markdown 文件：
- **导出完整文章** — 包含标题、作者、正文内容
- **方便分享存档** — 导出为标准 Markdown，可粘贴到任何笔记工具
- **AI 分析** — 导出的 Markdown 可直接用于 AI 工具分析和总结
- 在工具栏点击 📄 导出按钮使用

## 💬 评论查看

- 在阅读中按 **，** 键加载并浏览评论区
- 查看更多用户的观点和讨论
- 再次按 **，** 收起评论区

## 🛠️ 工具栏自定义

支持沉浸模式下的浮动工具栏，可自定义：
- **显示/隐藏** — 在设置面板中勾选要显示的按钮
- **顺序调整** — 拖拽按钮项调整排列顺序
- **快速隐藏** — 沉浸模式下悬停按钮点击 × 快速隐藏
- **恢复默认** — 一键恢复默认工具栏配置

## 🎨 设置面板功能

点击 **。** 键或设置按钮打开设置面板，包含 5 个 Tab：

### 🔤 文本样式
| 选项 | 说明 |
|------|------|
| 字体大小 | 8-24px 范围调整 |
| 行高 | 1.0-2.5 倍行间距 |
| 最大宽度 | 300-2400px 阅读宽度 |
| 字体选择 | 微软雅黑、宋体、楷体、等宽字体等 |
| 字体颜色 | 取色器 + 12 种常用颜色快选 |
| 透明度 | 0-100% 文字透明度 |
| 对齐方式 | 左/中/右/两端对齐 |

### 🎬 多媒体
- **3 种模式**：正常显示 / 迷你模式 / 全部隐藏
- **迷你模式缩放**：1-100% 精确控制
- 快捷键 **/** 循环切换 |

### ⭐ 功能增强
- **灰色模式** — 全页面灰色显示，防刺眼 + 更隐蔽
- **智能伪装** — 失焦自动伪装成代码文件
- **侧边栏伪装** — 同步伪装侧边栏为文件列表
- **伪装文件类型** — 自定义 JS/TS/Python/Java 等类型
- **关注列表过滤** — 隐藏「赞同了」类型的内容
- **回答内容过滤** — 仅展示未读的回答

### 🛠️ 工具栏
- 自定义按钮显示/隐藏
- 拖拽调整顺序
- 全选/全不选/恢复默认

### ⌨️ 快捷键
- 为每个功能按钮绑定自定义快捷键
- 支持 Ctrl/Alt/Shift 修饰键
- 一键恢复默认 / 清空所有快捷键

## 贴心功能

### 设置持久化
- 所有设置自动保存到 localStorage
- 重启 VSCode 后设置依然生效
- 支持恢复默认设置

### 实时预览
- 所有设置修改立即生效
- 无需刷新页面即可看到效果
