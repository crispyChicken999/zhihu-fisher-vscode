# 开始探索内容

点击侧边栏的 **📖 书本图标**，您会看到五个主要视图：

## 可用视图

### ⭐ 推荐
浏览基于您兴趣的个性化推荐内容
- 支持「不喜欢该内容」减少类似推荐
- 支持「不再推荐该作者」屏蔽作者
- 支持一键收藏到收藏夹

### 👥 关注
查看您关注的人的动态，包括：
- 回答/文章/想法等创作内容
- 关注人「赞同了」的内容（可过滤隐藏）
- 支持「不喜欢」、「屏蔽作者」、「收藏」操作
- 支持加载更多关注内容

### 🔥 热榜
实时获取知乎全站热门内容，掌握当下最火热的话题讨论

### 🔍 搜索
主动搜索您感兴趣的任何话题
- 支持搜索后收藏结果
- 支持重置搜索历史

### 📁 收藏
管理和浏览您的知乎收藏夹
- 我创建的收藏夹 / 我关注的收藏夹
- 直接在 VS Code 中创建、删除收藏夹
- 从推荐/关注/搜索中一键收藏内容

## 🔗 浏览指定链接

除了浏览列表内容，您还可以直接输入知乎 URL 打开任意内容：
- 支持 `zhuanlan.zhihu.com/p/xxx` 专栏文章
- 支持 `www.zhihu.com/question/xxx` 问题页
- 支持 `www.zhihu.com/pin/xxx` 想法页
- 在侧边栏顶部点击 🔗 链接图标即可输入

## 内容增强功能

### 回答内容过滤
在问题详情页中，可根据是否点击过「赞同/不赞同」来过滤回答：
- **全部展示** — 正常显示所有回答
- **仅展示未读** — 只显示未点击过赞踩的回答，快速找到新鲜内容

### 回答预加载数量
可自定义每批预加载的回答数量（5-50个），在侧边栏菜单或设置中调整。

## 使用提示
- 点击任意内容即可在内置阅读器中查看完整文章
- 阅读时支持回答排序切换（默认/时间排序）
- 支持 A/D 键快速切换上下回答
