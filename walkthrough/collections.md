# 知乎收藏夹功能

**知乎摸鱼**提供了完整的收藏夹管理功能，您可以直接在 VS Code 中浏览、收藏、创建和管理收藏夹！

## 📖 浏览收藏夹

收藏夹视图分为两大板块：

### 📂 我创建的收藏夹
- 查看自己创建的所有收藏夹
- 每个收藏夹内可浏览收藏的文章、回答
- 显示收藏总数、创建者、更新时间等信息

### 📂 我关注的收藏夹
- 查看关注的其他用户的收藏夹
- 同样支持浏览内部内容

## ⭐ 收藏操作

### 收藏内容
在 **推荐**、**关注**、**搜索** 列表中，每个内容项都提供收藏按钮，一键收藏到指定收藏夹：
1. 点击内容旁的 ⭐ 收藏按钮
2. 在弹出的选择器中选择目标收藏夹
3. 收藏成功！

### 取消收藏
- 在收藏夹内找到要移除的内容
- 右键 → 取消收藏 或 点击移除按钮
- 确认后即从收藏夹中移除

### 在浏览器中打开
- 支持在浏览器中打开整个收藏夹页面
- 也支持在浏览器中打开单个收藏项

## ⚙️ 管理功能

### 创建收藏夹
直接在 VS Code 中创建新收藏夹：
1. 在收藏夹视图中点击创建按钮
2. 输入名称（必填）、描述（可选）
3. 选择可见性（公开/私密）
4. 创建成功！收藏夹立即可用

### 删除收藏夹
- 可删除自己创建的收藏夹
- 删除前会弹出确认对话框，防止误操作
- 删除不可撤销，请谨慎操作

### 刷新与控制
- 🔄 **刷新全部** — 刷新所有收藏夹列表
- 🔄 **刷新我创建的** — 仅刷新我创建的收藏夹
- 🔄 **刷新我关注的** — 仅刷新我关注的收藏夹
- ⬇️ **加载更多** — 分页加载更多收藏夹/收藏项
- 🧹 **清理缓存** — 清理收藏夹选择器缓存，下次收藏时获取最新数据

## 💡 使用提示

- 收藏夹数据与知乎账号同步，换个设备也能看到
- 收藏时如果列表为空，请先刷新收藏夹列表
- 如果收藏夹列表没有显示，请先**扫码登录或设置Cookie**登录知乎

点击侧边栏中的 **收藏** 标签开始使用！
