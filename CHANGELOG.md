# 更新日志 (Changelog)

本文档记录了"知乎摸鱼"(Zhihu Fisher) VS Code 扩展的所有重要更改。

## [0.5.3] - 2025-11-18

### Features

- 调整了沉浸模式下内容中 h1-6 的字体大小，统一为设置的正文字体大小（1em），避免标题过大影响阅读体验。
  > 原因是，在看回答的时候看得好好的，切到下一个回答，回答里面突然一个 h1 特别大，吓一跳，特别明显，容易被抓包 😂。沉浸模式下就统一字号，如果想正常大小浏览的话就切回正常模式看吧（按 X 键）。

### Bug Fixes

- 修复了侧边栏伪装功能的条件判断，优化联动逻辑，确保在有任何知乎详情页打开时侧边栏保持伪装状态，只有全部关闭后才恢复正常显示。
- 修复了子评论弹窗样式异常的问题，使用 max-height 代替 height，动态适应高度。

## [0.5.2] - 2025-11-14

### Features

- 新增沉浸模式下作者信息透明度设置功能，用户可以在外观设置中调整作者信息的透明度，提升阅读体验。
  - 透明度范围从 1 到 100，用户可以根据个人喜好进行调整。
  - 透明度设置会实时生效，用户可以立即看到效果。
  - loading 界面也会应用透明度设置，确保整体视觉一致性。
- 优化了沉浸模式下作者信息的位置及样式，优先看回答，作者信息收拢到底部的信息栏中，提升了视觉效果和用户体验。
- 优化了关联问题弹窗中的 item hover 样式，提升了交互体验。
- 优化了评论区列表子评论作者信息的展示样式，一行显示更加美观。

## [0.5.1] - 2025-11-11

### Bug Fixes

## [0.4.8] - 2025-10-29

### Features

- 针对 win11 系统，Chrome 会自动使用 `效能模式/效率模式/Efficiency mode` 运行浏览器，导致 Puppeteer 爬取数据缓慢，新增了浏览器启动参数 `--disable-features=UseEcoQoSForBackgroundProcess`，强制禁用效能模式，确保在 win11 系统中能够流畅运行。（如有问题请反馈）
- 使用原生 `Fetch` 代替 `Axios` 进行网络请求，避免出现 `Axios` 在某些环境下无法正常工作的情况，提升了网络请求的稳定性和兼容性，同时减小了扩展包的体积。
- 优化了控制台输出，添加了评论区和子评论的请求 URL 及返回数据日志，方便调试和排查问题。
- 优化了收藏 Tooltip 的显示内容及样式，与推荐、热榜等侧边栏项保持一致，优化了逻辑处理减少冗余，提升了用户体验。
- 优化了刷新收藏列表的命令逻辑，添加了加载状态检查，在刷新列表（热榜、推荐、搜索、收藏）的时候不能刷新，避免冲突导致的卡住无法加载的问题。

### Bug Fixes

- 修复了相关问题回答的按钮在切换回答时候闪烁的问题，去掉了 `transition`，避免闪烁影响使用体验。
- 针对文章，去掉了相关问题的图标和按钮，避免误导，因为文章是没有相关问题的。
- 删除了备份的 `scripts-backup.ts` 文件，避免冗余文件影响维护（已重构，功能完全实现，不再需要此文件做参考）。
